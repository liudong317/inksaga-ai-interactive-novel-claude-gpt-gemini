# InkSaga 墨叙 · AI互动小说桌面游戏

> 软件名称：**InkSaga 墨叙**

**InkSaga 墨叙** 是一款 AI 互动小说桌面应用。配置角色与剧情，由大语言模型实时生成故事分支，支持多轮对话与记忆管理。

---

## 快速开始

```bash
git clone https://gitee.com/liudong317/inksaga-ai-interactive-novel-claude-gpt-gemini.git
cd inksaga-ai-interactive-novel-claude-gpt-gemini
npm install
npm start
```

## 模型配置（晴红API）

```yaml
Base URL: https://www.qinghong.tech/v1
Model:    deepseek-v4-pro
```

| 模型 | 说明 |
|------|------|
| `glm-5.2` | 智谱 GLM |
| `deepseek-v4-pro` | DeepSeek 推理 |
| `qwen3.7-max` | 通义千问 |
| `gpt-5.5` | GPT 系列 |
| `claude-opus-4-8` | Claude 旗舰 |
