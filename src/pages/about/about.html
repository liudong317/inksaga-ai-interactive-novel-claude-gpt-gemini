InkSaga 墨叙

v1.0.0

AI 互动小说 · 桌面创作引擎

软件简介

InkSaga 墨叙 是一款面向创作者与玩家的 AI 互动小说桌面应用。 你只需设定角色身份、剧情类别与世界观描述，大语言模型便会实时推演故事走向， 生成多分支叙事、角色对话与事件反馈，让每一次选择都推动剧情向前。

不同于简单的聊天 Demo，墨叙提供从基本配置 → 模型接入 → 剧情设定 → 事件编排 → 开始游戏 的完整工作流，并支持对话记忆与本地存档，适合长篇连载、跑团式叙事与创意写作实验。

核心功能

  • 角色与世界观配置 — 自定义姓名、性别、题材、人数与事件规模
  • 多模型自由切换 — 接入晴红API，支持 Claude / GPT / Gemini / DeepSeek 等
  • 剧情与事件编辑 — 可视化配置场景、人物与事件树
  • 实时 AI 叙事 — 多轮对话驱动，记忆上下文，沉浸互动
  • 本地数据安全 — 配置与聊天记录保存在本机，隐私可控

推荐使用流程

  1. 在「基本配置」填写角色与剧情描述
  2. 在「模型配置」接入晴红API 并测试连接
  3. 完善「剧情配置」与「事件配置」
  4. 点击「开始游戏」，进入 AI 互动叙事

推荐 AI 模型中转 · 晴红API

OpenAI 兼容接口，一个 API Key 即可调用多种主流大模型，按模型名称切换即可。
